# Develop: weekly review

Ask one section at a time:

- What meaningful decisions did you make?
- What did you ship, learn, or influence?
- What feedback did you receive?
- What evidence should enter the career bank?
- Where did AI help, mislead, or lack context?
- What deserves attention next week?

Detect potential evidence without duplicating existing records. Confirm claims and confidentiality before saving.

Call:

```bash
python -m pco review weekly --input <json-file>
```
