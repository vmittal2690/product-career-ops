# Search: build packet

Load the opportunity, scorecard, profile, and evidence. Create a grounded JSON packet source and run:

```bash
python -m pco packet build --id <OPP-ID> --input <json-file>
```

Generate:

- tailored ATS resume
- executive editorial cover letter
- role scorecard
- company brief
- interview packet
- outreach draft

No unsupported claim may enter a document.
